[pico-jxgLABO](https://zenn.dev/ypsitau/articles/2025-08-01-labo-intro) は、USB ケーブル一本でマイコンボード RaspberryPi Pico の様々な機能を試すことができる実験プラットフォームです。

pico-jxgLABO に OV7670 カメラモジュールのサポートを追加しました！

OV7670 はとてもポピュラーなカメラモジュールで、一個 500 程度で購入できる手軽さから多くの電子工作プロジェクトで利用されています。検索すると多くの工作例が見つかるのですが、験的な段階に留まっており、実用的に使うには工夫が必要です。

pico-jxgLABO では、プログラムをコーディングすることなく、コマンド操作で OV7670 カメラモジュールをセットアップし、すぐに使い始めることができます。

この記事では、まず OV7670 カメラモジュールと Pico ボード (+ ホスト PC) だけでできる以下の実験を紹介します。

- USB 接続したホスト PC に動画ストリーミングを行う
- カメラ画像を BMP 形式で Pico 内部のストレージに保存する

次に、TFT LCD ディスプレイモジュール ST7789 を接続して、以下の実験を行います。

- カメラ画像を Pico ボードに接続した TFT ディスプレイにリアルタイム表示する

なお、開発や動作確認をするホスト PC は Windows を想定しています。

## OV7670 カメラモジュールについて

OV7670 は、Omnivision が開発した VGA 解像度 (640x480) のカメラモジュールです。I2C インターフェースを使用してレジスタ設定を行い、クロックで動機を取りながら 8 ビットのパラレルデータバスで画像データを出力します。

![OV7670 カメラモジュール](https://raw.githubusercontent.com/ypsitau/zenn/main/images/2025-12-22-labo-ov7670/ov7670.jpg)

レジスタの仕様などを記載したデータシートは、開発元のサイトには見当たりません。僕が参照したのは以下のリンク先の PDF ファイルです。特に、Implementation Guide は、混沌としたレジスタ群を機能ごとにカテゴリー分けしており、動作を理解するための参考になります。

- [OV7670_2026](https://web.mit.edu/6.111/www/f2016/tools/OV7670_2006.pdf)
- [OV7670 Implementation Guide (V1.0)](https://www.haoyuelectronics.com/Attachment/OV7670%20+%20AL422B(FIFO)%20Camera%20Module(V2.0)/OV7670%20Implementation%20Guide%20(V1.0).pdf)

それでも、これらのドキュメントは完全ではなく、肝要な情報が欠落しています。特に、デフォルトのレジスタ設定ではモノクロ画像しか取得できず、カラー画像を取得するには「レジスタ `0xB0` に `0x84` を設定する」という操作が必要なのですが、これが何を意味するのかは不明です。実際に動作するコードを作成するには、有志が開発したプログラムを参考にしながら試行錯誤で理解を深める必要があります。

## pico-jxgLABO の書き込み

Pico ボードへの pico-jxgLABO の書き込みと基本的な使い方は[こちら](https://zenn.dev/ypsitau/articles/2025-08-01-labo-intro#pico-jxglabo-%E3%81%AE%E5%B0%8E%E5%85%A5%E6%96%B9%E6%B3%95)。特別なハードウェアは必要なく、Pico や Pico 2 ボードを USB ケーブルで PC に接続するだけで始められます。ターミナルソフトを使ってコマンドを入力し、Pico ボードを操作します。

この記事で説明する実験を行うには、バージョン **0.6.2** 以降の pico-jxgLABO が Pico ボードに書き込まれている必要があります。ターミナルソフトで `about-me` コマンドを実行すると pico-jxgLABO のバージョンを確認できます。

```text
L:/>about-me
Program Information
 name:              pico-jxgLABO
 version:           0.6.2
     :
     :
```

## OV7670 カメラモジュールの接続

OV7670 カメラモジュールは、I2C インターフェースと複数の GPIO ピンを使用して Pico ボードに接続します。以下の配線例を参考にしてください。

![OV7670 と Pico ボードの接続例](https://raw.githubusercontent.com/ypsitau/zenn/main/images/2025-12-22-labo-ov7670/ov7670-wiring.png)

接続する GPIO ピンはコマンドでカスタマイズができます。以下の点に留意してください。

- D0 から D7 までの 8 本のデータラインは連続した GPIO ピンに接続する必要があります
- SDA と SCL は Pico ボードの I2C0 またｈ I2C1 のいずれかの I2C インターフェースに接続します
- XCLK, PCLK, HREF, VSYNC は任意の GPIO ピンに接続できます

最も周波数の高い XCLK 端子には 24 MHz の信号が供給されるので、配線はできるだけ短くすることが望ましいです。とはいうものの、それほど神経質になる必要もないようで、手元の実験では 16 cm 程度のジャンパーワイヤーで接続して問題なく動作しています。

![OV7670 と Pico ボードの接続例](https://raw.githubusercontent.com/ypsitau/zenn/main/images/2025-12-22-labo-ov7670/ov7670-wiring-photo.jpg)
*OV7670 カメラモジュールと Pico ボードの接続例*

以下のコマンドを実行して、I2C インターフェースとカメラモジュールをセットアップします。

```text
L:>i2c0 -p 16,17 --baudrate:100000
L:>camera-ov7670 setup {i2c:0 d0:2 xclk:10 pclk:11 href:12 vsync:13}
```

- `i2c0` コマンドで I2C0 インターフェースを GPIO16 (SDA), GPIO17 (SCL) に設定し、通信速度を 100 kHz に指定しています
- `camera-ov7670 setup` コマンドで OV7670 カメラモジュールの各種接続ピンを指定しています。I2C インターフェースとして i2c0 を選択しています。D0 は GPIO2 に接続し、D1 から D7 までは連続した GPIO3 から GPIO9 に自動的に割り当てられます。XCLK, PCLK, HREF, VSYNC はそれぞれ GPIO10, GPIO11, GPIO12, GPIO13 に接続しています。

これで、OV7670 カメラモジュールが使用可能になります。`camera-ov7670` コマンドで `dump` サブコマンドを実行して、カメラモジュールのレジスタ設定を確認してみましょう。

### 動画のストリーミング

Windows PC で、タスクバーの検索ボックスに「カメラ」と入力して、カメラアプリを起動します。PC に複数のカメラデバイスが接続されている場合は、下図のように右上にカメラ選択ボタンが表示されるので、"pico-jxgLABO" が表示される画面に切り替えます。

![Windows のカメラアプリ](https://raw.githubusercontent.com/ypsitau/zenn/main/images/2025-12-22-labo-ov7670/windows-camera-app.png)

ターミナルで以下のコマンドを実行して、OV7670 カメラモジュールからホスト PC に動画ストリーミングを開始します。

```text
L:>camera transmit-video-start
```


### 画像のキャプチャ

```text
L:>camera capture:image.bmp
```


### カメラディスプレイの開始

ST7789 TFT ディスプレイモジュールを Pico ボードに接続します。以下の配線例を参考にしてください。

![ST7789 と Pico ボードの接続例](https://raw.githubusercontent.com/ypsitau/zenn/main/images/2025-12-22-labo-ov7670/ov7670-st7789-wiring.png)

以下のコマンドを実行して、TFT ディスプレイをセットアップします。

```
L:>spi0 -p 18,19,16 --baudrate:20000000
L:>display-st7789 setup {spi:0 dc:17 cs:16 rst:20 blk:21}
```

以下のコマンドを実行して、画面全体が赤色になれば接続成功です。

```text
L:>draw fill {color:red}
```

`display-start` サブコマンドを使用して、OV7670 カメラの映像を TFT ディスプレイにリアルタイム表示します。

```text
L:>camera display-start
```

### 各種パラメータの設定

ミラー表示

```text
L:>camera-ov7670 mirror:on
```

上下反転

```text
L:>camera-ov7670 vflip:on
```

明るさ調整

```text
L:>camera-ov7670 brightness:2
```

コントラスト調整

```text
L:>camera-ov7670 contrast:2
```
